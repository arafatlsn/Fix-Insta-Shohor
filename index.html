Q. How does JavaScript Works?

Ans.

JavaScript run on browser with JS engine. The Chrome Browser used V8 JS engine. The Mozilla used Spider-Monkey and Edge used Chakra JS engine. But it's takes few steps to running on browser.
1. First step is getting JavaScript code from the network.
2. V8 / Spider-Monkey / Chakra parses the source code and convert into Abstract Syntax Tree (AST).
3. The AST the Ignition interpreter can start to do its thing and produce bytecode.
4. At that point, the machine can understand the code and give output.

Q. what is the comparison between Local Storage and Session Storage?

Ans.

sessionStorage is similar to localStorage

The main difference between Local and Session storage is that :
Local Storage: While data in localStorage doesn't expire.

Session Storage: While data in sessionStorage is cleared when the page session ends.
